Even Better OneTab

Read-only viewer for your synced tab lists.

How do I set this up?
  1. In the extension on your computer, open Settings → Sync & mobile and connect sync (one-time).
  2. Use the Token and Gist id copy buttons there and get them onto this device — e.g. paste into a note or a message to yourself.
  3. Enter them below. They're stored only in this browser, never sent anywhere except to GitHub itself.

Tip: use your browser's Add to Home Screen on this page to make it feel like an app.
